---
title: 'Download and Install Kiro IDE'
---

## Download

1. Open a browser → [https://kiro.dev/downloads/](https://kiro.dev/downloads/)
2. Under **Windows (x64)**, click the download button
3. Save the installer to `Downloads`

## Run the Installer

1. Open **Downloads** → double-click the Kiro installer
2. Follow the on-screen prompts (accept license, choose install location if asked)
3. If offered, check **Add to PATH** (or **Add "kiro" command to PATH**) — required if you want to install extensions from PowerShell in the next section
4. Click **Install** → wait for completion → **Finish**
5. Launch **Kiro IDE** from the Start menu
