## Open Your Workspace

If the folder does not exist yet, create it from the WSL terminal:

```bash
mkdir -p ~/workspace
```

Then in Kiro: **File → Open Folder** → `/home/<your-username>/workspace` → **Yes, I trust the authors** when prompted:

![Kiro workspace trust prompt for the workspace folder](/kiro-wsl-aws-setup-guide/images/kiro-workspace-trust.png)
