---
title: "Install AWS CLI"
---

Install the AWS CLI v2 if not already present:

```bash
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
rm -rf aws awscliv2.zip
```

Verify the installation:

```bash
aws --version
```

**Expected:** A version string like `aws-cli/2.x.x Python/3.x.x Linux/...`
